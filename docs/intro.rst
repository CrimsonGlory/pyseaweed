============
Intro
============

Python module to communicate with Weed-FS_.


============
Installation
============

From PyPI::

    pip install pyweed


From GitHub::

    pip install git+https://github.com/utek/pyweed.git


From source::

    git clone https://github.com/utek/pyweed.git
    cd pyweed
    python setup.py install







.. _Weed-FS: http://code.google.com/p/weed-fs/
